# Enterprise E-Commerce Landing Page

A modern, Netflix-inspired e-commerce landing page built with vanilla HTML, CSS, and JavaScript.

## Features

✨ **Modern Design**
- Dark theme with red accents (Netflix-style)
- Minimalist and premium aesthetic
- Smooth animations and transitions

🛍️ **E-Commerce Functionality**
- Product filtering by category
- Shopping cart with local storage
- Responsive product grid

📱 **Responsive Design**
- Mobile-first approach
- Works perfectly on all devices
- Smooth mobile navigation

## Getting Started

1. Open `src/index.html` in a browser
2. Or use a local server:
```bash
cd src
python3 -m http.server 8000
```

Visit `http://localhost:8000` in your browser.

## Technologies Used

- HTML5

- LocalStorage API
